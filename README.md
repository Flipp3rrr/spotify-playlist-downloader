# spotify-playlist-downloader
A tool to download playlists from Spotify from YouTube
## Requirements
* spotipy
* youtube_dl (`pip3 install -U youtube_dl`)
* a client ID, client secret, and spotify toekn (from `developer.spotify.com`)
